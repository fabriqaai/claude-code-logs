# Bolt: 011-client-renderer

## Overview

Create an HTML shell with embedded JavaScript that loads Markdown files via fetch, renders them using marked.js + highlight.js, and provides download/copy buttons. Preserves the existing UI/UX.

## Objective

Enable client-side Markdown rendering while maintaining identical visual appearance and adding download/copy functionality.

## Stories Included

- **001-md-rendering**: Client-Side Markdown Rendering (Must)
- **002-download-button**: Download as Markdown Button (Should)
- **003-copy-button**: Copy as Markdown Button (Should)
